I made this page for myself. it generates my release catalog numbers - if you don't know what those are, don't worry, you don't need to. they're essentially just strings of text/numbers that are shorthand for certain things I've made. for example, KA00-08 refers to my song "nihilism" off the album "written in stone". these numbers are unique, so if I ever release a song that has multiple versions, these numbers will specify which one. you will never need to use these, and nobody (except me internally) will ever refer to my stuff with the full codes. however, I sometimes do use shorter versions of these codes to refer to things: e.g. albums "KA01" or gamedev projects e.g. "KG02"



there's a set of rules for how these codes work. basically, you stack the parts you're referring to with the codes left to right, biggest to smallest. for example, if the code is talking about a specific part of a song, the album comes first, then the song, then the part of that song.
